Detailed Description

Advanced ruleset with elevated difficulty and winning threshold of 25.

Implements challenging gameplay conditions including 2-deck shoe, 3 initial cards per player, and dealer standing at 21. Provides higher risk and reward dynamic compared to standard rules.
