\doxysubsection{Detailed Description}
Base class defining the core rules and configuration for Blackjack games. 

This class provides default game rules and settings that can be customized through inheritance. It defines the fundamental parameters that control game behavior.\hypertarget{class_base_rules_game_end_conditions}{}\doxysubsection{\texorpdfstring{Game End Conditions}{Game End Conditions}}\label{class_base_rules_game_end_conditions}

\begin{DoxyItemize}
\item A player loses when they run out of money
\item The dealer loses when they run out of money
\item The game ends when either the dealer loses or all players lose
\end{DoxyItemize}\hypertarget{class_base_rules_player_ranking}{}\doxysubsection{\texorpdfstring{Player Ranking}{Player Ranking}}\label{class_base_rules_player_ranking}
Players are ranked in the following order\+:
\begin{DoxyItemize}
\item First\+: Players still at the table at the end of the game, ranked by money amount
\item Second\+: Players who have lost, in reverse order of elimination (last eliminated first) 
\end{DoxyItemize}
