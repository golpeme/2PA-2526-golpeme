\doxysubsection{Detailed Description}
Interface representing the Blackjack game table and its operations. 

This interface provides access to game state information and methods to manipulate the game flow. It handles player hands, bets, card dealing, and round management.

\begin{DoxyNote}{Note}
The dealer is part of the table and has no choices beyond applying the rules. 
\end{DoxyNote}
