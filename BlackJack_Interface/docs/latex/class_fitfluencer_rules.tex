\doxysubsection{Detailed Description}
Standard ruleset with reduced winning threshold of 20. 

Classic single-\/deck blackjack with traditional 2 initial cards and soft 17 rule (dealer hits on soft 17, stands on hard 17+). Most closely resembles standard casino blackjack rules.
